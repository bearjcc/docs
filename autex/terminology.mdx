This page keeps a human-facing subset for quick onboarding.

| Term | Definition |
|---|---|
| DriveWorks Group | Data container for projects, tables, security, and specifications. |
| Project | Container for forms, rules, and outputs stored in a DriveWorks Group. |
| Specification | One run of a project that tracks outputs through its lifecycle. |
| DW Tools | Toolkit command group shown in DriveWorks Administrator. |
| Autex Tools | Toolkit window for running scripts and viewing script output. |
| Release script | Supported user-facing script under `DWPlugin/Scripts/Release`. |
| Dev script | Internal diagnostic or probe script under `DWPlugin/Scripts/Dev`. |
| Test script | Verification script under `DWPlugin/Scripts/Test`. |
| WIP script | Work-in-progress script under `DWPlugin/Scripts/WIP`; not release-ready. |
| CPQ | Configure, Price, Quote. |
| SQL | Structured Query Language. |
| CRM | Customer Relationship Management. |
