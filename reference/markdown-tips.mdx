## Images

Use media paths rooted at `/images/wiki/`.

```markdown
![Administrator: default specification and group content folder paths](/media/administrator-spec-group-content-folder-paths.png)
![Same asset constrained to 64px width](/media/administrator-spec-group-content-folder-paths.png =64x)
```

## Tabs

Use tabsets for alternative views.

~~~markdown
## Example {.tabset}

### Preview
Rendered content

### Source
```text
raw markdown
```
~~~

## Blockquotes

Use style classes on the next line.

```markdown
> Informational text.
{.is-info}

> Warning text.
{.is-warning}
```

## Details Blocks

Use `<details>` for archived or optional content.

```html
<details>
<summary>Expand</summary>

Optional content.

</details>
```

## Lists

```markdown
- Bullet item
- Another item

1. Numbered item
2. Numbered item
```

## Code Blocks

SELECT * FROM dbo.Specifications;

~~~

## DriveWorks rule examples

Use the `driveworks` language tag for **custom function** calls copy-pasted into **Rule Builder**:

~~~markdown
### Example

```driveworks
AutexSlabPackingImage(DwLookupNested,1,DWLookupSlabPackingSettings,TRUE)
```
~~~
