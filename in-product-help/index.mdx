Contextual help should deep-link into this **Mintlify** site from Toolkit script descriptions, error strings, and Administrator surfaces. Search (**Ctrl+K** / **Cmd+K**) is the primary entry when no dedicated link exists.

## Wiki.js cutover

**Wiki.js** is the legacy site. New and updated docs belong in `mintlify/` only. When Mintlify is the team default:

1. Freeze Wiki.js edits except redirects.
2. Point bookmarks and `/home` to the Mintlify home page.
3. Add Toolkit UI links to stable Mintlify paths (not repo-relative paths).
