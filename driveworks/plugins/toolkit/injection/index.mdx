## Administrator injection guides

These pages document how the Autex Toolkit plugin extends **DriveWorks Administrator**: command bars, sidebar, Rules Builder, Form Designer, property grid, and floating windows. Each guide names the shipping example, entry files under `Autex Toolkit/DWPlugin/`, and verification steps.

Operator-facing "where does Toolkit appear?" lives in [Administrator surfaces](/driveworks/plugins/toolkit/administrator-surfaces).

Maintainer-facing **region names and injection targets**: [Administrator UI region map](/driveworks/plugins/toolkit/injection/administrator-ui-map).

| [Adding a QuickBar Button](/driveworks/plugins/toolkit/injection/add-quickbar-button)                                 | Title-bar QuickBar (global)                      | SDK `IQuickBarManager` + app `ICommandManager` | Autex Toolkit (open Autex Tools)         |
| [Adding a View-Scoped Command Button](/driveworks/plugins/toolkit/injection/add-view-scoped-command-button)           | Command bar on one view + dynamic enablement     | SDK commands + reflection selection probe      | Macro triggers                           |
| [Adding a Virtual Split Command Button](/driveworks/plugins/toolkit/injection/add-virtual-split-command-button)       | Command bar dropdown                             | SDK `IVirtualSplitCommandButton`               | Form Designer Convert                    |
| [Adding a Left Sidebar Menu Item](/driveworks/plugins/toolkit/injection/add-sidebar-menu-item)                        | Left ToDo tree / content host                    | Reflection `InsertControlViewAfter`            | Group Constants                          |

| [Adding a Form Designer Context Menu Item](/driveworks/plugins/toolkit/injection/add-form-designer-context-menu-item) | Form Designer right-click                        | Reflection WPF `ContextMenu`                   | Inspect, Create Metaclass, Add Metaclass |
| [Adding Form Control Properties](/driveworks/plugins/toolkit/injection/add-form-control-properties)                   | Property grid + optional WebView bridge          | `DynamicProperty.Register` + grid wrap         | Layout Grid, Metadata                    |

## Choose a pattern

Not sure which chrome region to extend? Start with the [Administrator UI region map](/driveworks/plugins/toolkit/injection/administrator-ui-map).

| Your feature needs…                                           | Start here                                                                                                   |
| ------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------ |
| A small icon in the title-bar QuickBar (always visible)       | [QuickBar Button](/driveworks/plugins/toolkit/injection/add-quickbar-button)                                 |
| A toolbar button everywhere Administrator shows a command bar | [Command Manager Button](/driveworks/plugins/toolkit/injection/add-command-manager-button)                   |
| A button on one view, enabled only with valid selection       | [View-Scoped Command Button](/driveworks/plugins/toolkit/injection/add-view-scoped-command-button)           |
| A toolbar dropdown whose items depend on selection            | [Virtual Split Command Button](/driveworks/plugins/toolkit/injection/add-virtual-split-command-button)       |
| A persistent pane in the left tree                            | [Sidebar Menu Item](/driveworks/plugins/toolkit/injection/add-sidebar-menu-item)                             |
| An action while editing a Rule                                | [Rules Builder Toolbar Button](/driveworks/plugins/toolkit/injection/add-rules-builder-button)               |
| An action on selected form controls in the designer           | [Form Designer Context Menu Item](/driveworks/plugins/toolkit/injection/add-form-designer-context-menu-item) |
| A new persisted slot on a control in the property grid        | [Form Control Properties](/driveworks/plugins/toolkit/injection/add-form-control-properties)                 |
