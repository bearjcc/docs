**Autex Toolkit** is the main Autex plugin for DriveWorks Administrator. It adds the **Autex Toolkit** / **DW Tools** command group, **Autex Tools** scripts, a **Group Constants** sidebar under **Group Setup**, Rule Builder helpers, Form Designer tools, and Project comparison workflows.

Use this hub if you operate or implement DriveWorks Projects with Toolkit installed. Programmer and build details are kept behind the advanced page.

## Safety rules

- Work in a development Group or Project for bulk changes.
- Make a **Pack and Go** or approved backup before applying mutating scripts.
- Read script danger levels before using **Run**.
- Remember that **Constants**, **Group Constants**, **Simple Tables**, and **Group Tables** store literal data, not Rule Builder formulas.
- Verify user-facing forms, Rules, and generated outputs after changes.
