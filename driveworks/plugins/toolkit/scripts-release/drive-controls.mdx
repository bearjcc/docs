**Drive controls** identifies input form controls, adds them as rows in a lookup or Calc table, and creates a **SetDefaults** Specification Macro that drives default values from the table.

**Danger level:** Writes on Run.

## Requirements

| Requirement  | Value        |
| ------------ | ------------ |
| Mode         | Project only |
| Open Project | Required     |

## Inputs

| Input                  | Default      | Description                                                 |
| ---------------------- | ------------ | ----------------------------------------------------------- |
| Table type             | Lookup table | Lookup (PowerPack task) or Calc table                       |
| Copy existing defaults | unchecked    | Copy each control's current default into the Default column |

## Output

Creates the DriveControls table, **SetDefaults** macro, and optionally the **Drive Default Controls from Table** task. Session log confirms each step.
