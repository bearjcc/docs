**Just fix it** embeds a standard YouTube iframe in the results WebView2 panel. It verifies embedded media in Toolkit UI; it does not change Project or Group data.

**Danger level:** Read-only.

## Requirements

| Requirement  | Value        |
| ------------ | ------------ |
| Mode         | Any          |
| Open Project | Not required |

## Inputs

None.

## Output

Video embed in the results area (autoplay muted; use player controls for sound).
