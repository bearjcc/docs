**Format all rules** runs the DriveWorks rule formatter over all rule-driven items and writes formatted Rules back. Covers Constants (value as formula), Variables, Model Rules, Calc Table Rules, and form control property Rules.

**Danger level:** Writes on Run.

## Requirements

| Requirement  | Value        |
| ------------ | ------------ |
| Mode         | Project only |
| Open Project | Required     |

## Inputs

None.

## Output

Session log shows the count of formatted Rules. Rules update in place immediately.
