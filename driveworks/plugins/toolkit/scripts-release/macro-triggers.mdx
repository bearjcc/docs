**Macro triggers** lists Specification Macros, then shows every trigger for the macro you select: form controls (OnChange, macro button), **Run Macro** tasks, and Specification Flow.

**Danger level:** Read-only.

## Requirements

| Requirement  | Value        |
| ------------ | ------------ |
| Mode         | Project only |
| Open Project | Required     |

## Workflow

1. **Run** — lists Specification Macros (Name, Category).
2. **Review** — select one macro (radio column), click **Show triggers**.
3. **Results** — table of trigger locations (**Source type**, **Location**).

## Inputs

None beyond selecting a macro in the review grid.
