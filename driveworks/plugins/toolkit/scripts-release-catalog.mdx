## Release scripts catalog

This catalog lists the Autex Toolkit Release scripts found in the current Toolkit source. Use it as an end-user guide for what each script does before running it in DriveWorks Administrator.

Danger levels are practical operating guidance: **Read-only**, **File export**, **Preview then apply**, or **Writes on Run**.

## Catalog

| Script | Purpose | Typical inputs | Danger level | More detail |
|--------|---------|----------------|--------------|-------------|
| **Just fix it** | Opens a quick troubleshooting video in the results panel. | None. | Read-only | `Autex Toolkit/docs/scripts/Release/JustFixIt.md` |
| **Find and replace** | Finds text in Rules and static values, then previews replacements before applying. | Find text, replacement text, data type/filter options. | Preview then apply | `Autex Toolkit/docs/scripts/Release/FindReplaceInRules.md` |
| **Unreferenced data** | Finds cleanup candidates such as unreferenced Constants, Variables, Simple Tables, Calc Tables, and Specification Macros. | Data type, indirect-reference option, cross-project option. | Preview then apply delete | `Autex Toolkit/docs/scripts/Release/UnreferencedData.md` |
| **List data** | Lists DriveWorks data by type for inspection and filtering. | Data type, name/rule/value filters, optional form control type filter. | Read-only | `Autex Toolkit/docs/scripts/Release/ListData.md` |
| **Check names (PascalCase & spelling)** | Checks names for PascalCase and spelling issues, then applies selected renames. | Data type, PascalCase checkbox, spelling checkbox, diagnostics checkbox. | Preview then apply rename | `Autex Toolkit/docs/scripts/Release/CheckNames.md` |
| **Compare projects** | Compares newer and older Project sources and shows an HTML diff report. | Source 1 type/path, Source 2 type/path. | Read-only | `Autex Toolkit/docs/scripts/Release/CompareProjects.md` |
| **Create data** | Creates one DriveWorks item of the selected type. | Data type, name, value/rule options, and type-specific fields. | Writes on Run | `Autex Toolkit/docs/scripts/Release/CreateData.md` |

## Extra cautions

**Writes on Run** scripts do not use the usual preview-only **Run** step. Use them only after testing on a disposable Project or Group copy.

**Group mode** can open, modify, save, and close multiple Projects depending on the script. Confirm the target Group and Project list before applying.

**Delete** actions should be treated as irreversible unless you have a fresh backup or source-controlled snapshot.
