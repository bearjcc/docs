This catalog lists Autex Toolkit **Release** scripts in Autex Tools. Check **Danger level** before **Run**; mutating scripts need a development Group or Project copy and a current backup.

Danger levels: **Read-only**, **File export**, **Preview then apply**, or **Writes on Run**.

## Catalog

| **Find and replace**                    | Finds text in Rules and static values, then previews replacements before applying.         | Find text, replacement text, data type/filter options.               | Preview then apply              | [Find and replace](/driveworks/plugins/toolkit/scripts-release/find-and-replace)                        |

## Extra cautions

**Writes on Run** scripts do not use preview-only **Run**. Test on a disposable Project or Group copy first.

**Group mode** can open, modify, save, and close multiple Projects. Confirm the target Group and Project list before applying.

**Delete** actions are irreversible without backup or source-controlled snapshots.
