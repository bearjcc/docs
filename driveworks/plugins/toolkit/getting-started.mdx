Open the **Autex Toolkit** command group in **DriveWorks Administrator** and launch **Autex Tools** — the window for running scripts and most Toolkit features. Toolkit ships as a DLL that IT installs for your environment; you work entirely from Administrator, not from the code repository.

<Steps>
## Open your Group

In **DriveWorks Administrator**, connect to the correct **Group**. Open a **Project** as well if your work is Project-scoped.

## Find the command group

Look for the **Autex Toolkit** / **DW Tools** command group on the command bar, or the Autex icon on the title-bar QuickBar.

## Launch Autex Tools

Select **Autex Toolkit** to open the **Autex Tools** window.

<Check>The Autex Tools window opens with the script browser and session log.</Check>
</Steps>

<Note>
No command group? Confirm the installed DLL and DriveWorks plugin registration with IT. See [Build and deploy](/driveworks/plugins/toolkit/build-and-deploy) for how the plugin loads.
</Note>
