## Getting started

**Autex Toolkit** runs inside DriveWorks Administrator. It adds tools for scripts, Group Constants, Rule Builder help, Form Designer styling, and project comparison.

IT installs the Toolkit DLL for your DriveWorks environment. Operators and implementers normally start from Administrator, not from the code repository.

## First check

1. Open **DriveWorks Administrator**.
2. Connect to the correct **Group**.
3. Open a **Project** if you are running Project-scoped work.
4. Look for the **Autex Toolkit** / **DW Tools** command group.
5. Open **Autex Toolkit** to launch **Autex Tools**.

If the command group is missing, confirm the installed DLL and DriveWorks plugin registration with IT.

## Before changing data

- Prefer a development copy of the Group or Project.
- Make a **Pack and Go** or approved backup before bulk changes.
- Read the script or feature page before applying changes.
- Treat **Writes on Run** scripts as high-risk.
- Re-test affected Rules, User Forms, and outputs after applying changes.
