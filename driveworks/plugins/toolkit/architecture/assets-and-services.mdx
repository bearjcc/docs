Shared **assets** and small **services** consumed across Toolkit UI and Rules Builder features — not feature screens themselves.

## In this layer

- **Icons** — `ToolkitIconLoader`, semantic icon map JSON, catalog providers for pickers.
- **Spelling** — Hunspell seed files under `Resources/SpellingSeeds/` (custom words, acronyms, mappings).
- **Export & utilities** — `CsvExportService` and similar one-purpose services.
- **Embedded resources** — theme and asset files referenced from multiple features.

Boundary: spelling *logic* and UDF registration sit in [Core Services](/driveworks/plugins/toolkit/architecture/core-services); picker UI sits in [Rules Builder Integration](/driveworks/plugins/toolkit/architecture/rules-builder) and [WebView UI](/driveworks/plugins/toolkit/architecture/webview-ui).
