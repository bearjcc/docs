The largest layer by file count. **Core Services** holds domain logic that transforms, compares, caches, and serializes **Project** data — the engine behind project compare, snapshot export, spelling, and script infrastructure.

## In this layer

- **Comparison** — `ProjectDataComparisonEngine`, change trees, typed comparers, known-limitation tagging for parity work.
- **Snapshots & TOML** — serialize/deserialize **Project** snapshots for git-based version control workflows.
- **Script hosting** — `AutexScript`, `AutexScriptContext`, registry, cancellation, and shared project extender UDF registration.
- **Cross-cutting utilities** — caching, rule formatting helpers, form-control analysis, diff summarizers.

Boundary: DTO shapes live in [Data Models](/driveworks/plugins/toolkit/architecture/data-models); UI that renders comparison or script tools lives in [WebView UI](/driveworks/plugins/toolkit/architecture/webview-ui) and [WPF Views](/driveworks/plugins/toolkit/architecture/wpf-views).
