## Color catalog lookups (Pantone and RAL)

Embedded catalogs ship in **AutexCustom** (no Group table required). Rule Builder category: **Autex Custom Functions**. Successful lookups return hex as `#RRGGBB` (e.g. `#FF0000`). Lookup failures return Excel-style error strings (`#INVALIDINPUT!`, `#KEY!`, `#VALUE!`); near-miss Pantone/RAL codes may include a **did you mean** suggestion on `#KEY!`.

<CustomFunctionDoc
  name="AutexGetPantoneHex"
  availability="Function"
  inputs="`pantone` (String). PMS number or catalog name (e.g. `186`, `Cool Gray 5`, `Process Yellow`). Optional leading `Pantone ` prefix (case-insensitive)."
  output="String, hex `#RRGGBB` for the match, or an error string"
  example="AutexGetPantoneHex(186)"
/>
