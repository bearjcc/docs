## Autex Custom Functions (Table By Column Name)

Same semantics as the native table functions they wrap; column arguments are **header text on row 1** (1-based “first data row is row 2” in the descriptions in code). Category in Rule Builder: **Autex Custom Functions (Table By Column Name)**.

  inputs="`table` (Table); `columnName` (String); optional `sortAscending` (Boolean)"
  output="Table (ListAll family)"
