When DriveWorks Administrator reports no license available, the floating license is usually held or the network license service has stalled. Restarting the service from DriveWorks License Manager clears it.

## License model

- One DriveWorks Administrator license per user session.
- DriveWorks Live concurrent usage is capped by available licensing capacity.

## No license available

<Steps>
## Rule out the simple causes

Confirm the license has not expired, and that another user is not currently holding the floating license.

## Restart the network license service

Open **DriveWorks License Manager**, go to the **Network License Server** tab, click **Stop Service**, then click **Start Service**.

## Retry

<Check>DriveWorks Administrator opens with a license.</Check>
</Steps>
