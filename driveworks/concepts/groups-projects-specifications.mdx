## Group

A **Group** is the DriveWorks database: Projects, security, captured components, Group tables, and shared assets. Autex production uses **AUTEX GLOBAL**; development uses **AUTEX DEV** (see [environment map](/driveworks/structure)).

## Project

A **Project** (`.driveprojx`) holds forms, Rules, Specification Flow, documents, and component sets. Authors edit Projects in **DriveWorks Administrator**.

## Specification

A **Specification** is one run of a **Project** — a configured order or quote instance. Live and Autopilot execute Specification Flow against released data.

## Modes in Toolkit

| Mode    | Scope                                  |
| ------- | -------------------------------------- |
| Project | Active open Project                    |
| Group   | Group-level data or loops all Projects |

Confirm mode before **Writes on Run** scripts. See [Autex Tools window](/driveworks/plugins/toolkit/autex-tools-window).
