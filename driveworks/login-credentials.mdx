## DriveWorks Data Server

Holds DriveWorks Group data, including Projects, specifications, and captured assets.

## Access

1. Open RTP (Remote Desktop, shortcut should be on desktop).
2. Use the **DriveWorks Data Server** entry in the team password manager (server `AZDWAPP01`, service account `Driveworks`).

## DriveWorks Live Server

Holds styling and static assets for:

- [https://driveworks.autex.co.nz/](https://driveworks.autex.co.nz/)
- [https://driveworksdevelopment.autex.co.nz/index.html](https://driveworksdevelopment.autex.co.nz/index.html)

## Access

1. Remote into the DriveWorks Data Server.
2. Open RTP (Remote Desktop) from that server.
3. Use the **DriveWorks Live Server** entry in the team password manager (server `AZDWLIVE01`, account `DriveWorks`).
